Trade Timer

Zeigt, wie lange die aktuelle Position offen ist — mit Farbwechsel ab einer Ziel-Haltedauer. Rein informativ.

Was der Indikator macht

Ein kompaktes HUD zeigt die Haltedauer der offenen Position im Format MM:SS:

  • Rot = unter der Ziel-Sekundenzahl
  • Grün = über der Ziel-Sekundenzahl
Gedacht als Hilfsmittel für Prop-Firm-Regeln (z.B. Mindest-/Ziel-Haltedauer): auf einen Blick sehen, ob eine Position lange genug offen war.

Wie Entry / Exit / Flip erkannt werden

Über das signierte Position.Volume (Long > 0, Short < 0, Flat = 0):

  • Entry aus Flat → Timer startet (Zeitstempel = Marktzeit).
  • Pyramiding / Teilschließung (gleiche Richtung) → Timer läuft weiter (erster Entry bleibt).
  • Flip (Long↔Short ohne Flat, Vorzeichenwechsel) → Timer wird neu gestartet.
  • Flat (Position zu) → Timer aus.

Als Zeitquelle dient die Marktzeit (candle.LastTime), nicht die Wanduhr — damit korrekt im Replay und unabhängig von UI-Hängern.
